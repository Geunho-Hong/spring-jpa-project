# spring-jpa-project
SpringBoot 기반의 JPA를 구현한 Project
* JPA를 학습하며 배웠던 내용을 바탕으로 1인 Toy 프로젝트 진행 
* 현재 개발 Developing 

* 주요 개발 내용 (향후 추가 예정)
  * ResponseEntity & Restful API 방식으로 View 없이 설계
  * 회원 가입 서비스
  * 게시글 CRUD 기능
  * 각 영역에 대한 TEST 코드 작성

* 사용 기술
  * Java
  * Spring Boot
  * JPA , Hibernate
  * Gradle
  * Junit5, Mokito, AssertJ 
  * IntelliJ
  * MySQL
